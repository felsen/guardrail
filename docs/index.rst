guardrail: object-level permissions for all
=========

**guardrail** is an ORM-agnostic implementation of object-level permissions.
**guardrail** currently supports SQLAlchemy, Peewee, Pony ORM, and Django ORM,
and can be extended to work with arbitrary ORMs and web frameworks.
